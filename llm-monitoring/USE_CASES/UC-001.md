# UC-001: GPU Data Collection

## Characteristic Information
**ID:** UC-001  
**Name:** GPU Data Collection  
**Priority:** High  
**Actor:** System (background)

## Description
Collect GPU metrics including VRAM usage, temperature, and fan speed using nvidia-smi.

## Main Success Scenario
1. System executes `nvidia-smi --query-gpu=memory.used,memory.total,temperature.gpu,fan.speed --format=csv,noheader,nounits`
2. Parses output line for each GPU
3. Returns structured data with:
   - VRAM used (MB)
   - VRAM total (MB)
   - Temperature (°C)
   - Fan speed (%)
4. Handles multiple GPUs if present

## Extensions
- 1a. nvidia-smi not available → return error, show message
- 1b. No NVIDIA GPU detected → show message, continue with other metrics
- 1c. Command fails → return None, don't crash application

## Sub-Variations
- Single GPU configuration
- Multi-GPU configuration (list all GPUs)

## Related Information
- Command: `nvidia-smi --query-gpu=memory.used,memory.total,temperature.gpu,fan.speed --format=csv,noheader,nounits`
- Output format: comma-separated values, one line per GPU
- Example: `4096,16384,65,30`
