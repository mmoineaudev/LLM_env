# UC-003: Git Metrics Collection

## Characteristic Information
**ID:** UC-003  
**Name:** Git Metrics Collection  
**Priority:** High  
**Actor:** System (background)

## Description
Collect Git repository metrics including uncommitted lines, commit status, and recent commits.

## Main Success Scenario
1. Detect if current directory is a Git repository
2. If yes, collect:
   - Uncommitted lines (added/deleted)
   - Commit status (dirty/clean)
   - Time since last commit
   - Recent commit messages (last 5)
3. Return structured data

## Extensions
- 1a. Not a Git repository → return None or empty data
- 1b. Git command fails → show error, don't crash

## Sub-Variations
- Repository with uncommitted changes
- Clean repository (no changes)
- Repository with staged and unstaged changes
- Different commit histories

## Related Information
- `git diff --stat` for uncommitted lines
- `git status` for dirty/clean state
- `git log -1 --format=%ai` for last commit time
- `git log -5 --format="%s - %ci"` for recent commits
- Current working directory context
