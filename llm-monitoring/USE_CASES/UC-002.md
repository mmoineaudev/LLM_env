# UC-002: RAM Data Collection

## Characteristic Information
**ID:** UC-002  
**Name:** RAM Data Collection  
**Priority:** High  
**Actor:** System (background)

## Description
Collect system RAM usage using /proc/meminfo.

## Main Success Scenario
1. Read /proc/meminfo file
2. Extract MemTotal and MemAvailable values
3. Calculate used percentage
4. Returns structured data with:
   - Total RAM (MB)
   - Available RAM (MB)
   - Used percentage

## Extensions
- 1a. /proc/meminfo not accessible → return error, show message
- 1b. Missing fields → use fallback calculation (MemTotal - MemFree)

## Sub-Variations
- Standard Linux /proc/meminfo format
- Different unit formats (kB, MB)

## Related Information
- File: /proc/meminfo
- Key fields: MemTotal, MemAvailable (or MemFree + Slab)
- Units: typically kB, convert to MB
